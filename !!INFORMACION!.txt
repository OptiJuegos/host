-Me sale un error al iniciar ¿Como lo soluciono?
Es posible que no te inicie por varios de estos motivos:

*Que los drivers de tu tarjeta grafica no esten actualizados y no puedan abrir el juego. Prueba actualizarlos o ejecutar el modo "OpenGL" o "Software"
*Que tu antivirus este bloqueando la ejecucion del juego y haya eliminado algunos archivos necesarios para abrirlos. Si es asi intenta deshabilitarlo y descomprimir el juego nuevamente
*Que tu sistema operativo sea tan viejo que no soporte el juego, como Windows 2000 o XP, si tienes windows 7 porfavor instala las ultimas actualizaciones
*Que el archivo este corrupto y los datos estan dañados. Porfavor intenta descomprimirlo utilizando la ultima version de 7-Zip y descargalo usando Free Download Manager
*Que el archivo se encuentre en muchas carpetas con nombres largos. Prueba mover la carpeta al escritorio
*Que el script .BAT hecho por mi no funcione bien en tu PC y de errores (muy raro que pase) si es asi intenta editarlo utilizando ChatGPT para intentar solucionar el problema
*Que no tengas las dependencias necesarias para ejecutar juegos instaladas, descargalas aqui: https://www.mediafire.com/file/2bjxzf59duumsr5/

-¿Que significa "NORMAL" y "OPTIMIZACION"? ¿Como inicio el juego?
Muy facil, el "1- (juego) NORMAL.bat" inicia el juego sin aplicar optimizaciones a windows
el "2- (juego) OPTIMIZACION.bat" inicia el juego aplicando optimizaciones a windows para obtener mejor desempeño.

Puedes iniciar el juego ejecutando el ".bat" que gustes
Si el juego no trae ".bat" inicia el ".exe"
