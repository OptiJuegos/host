SCHEDULE LITE

Bienvenido al proyecto de SCHEDULE LITE! Una version modificada, pensada para aumentar el rendimiento en computadoras de bajos recursos

¿En que version de SCHEDULE esta basada?

Esta basada en la version ultima version de steam del juego, con algunos parches de optimizacion y traducido al español

¿Cuanto pesa el juego?

El juego pesa 1.6 GB comprimido y 7.1 GB descomprimido.

⚠️¿Por que hay archivos corruptos en el juego y por ende no me inicia?

Este problema suele ocurrir cuando la descarga se realiza con una conexión a internet inestable. Recomendaciones:

  1. - Utiliza gestores de descarga como Free Download Manager o Internet Download Manager para garantizar una descarga completa.
  2. - Descomprime el archivo utilizando la última versión de 7-Zip para evitar problemas de extracción.

¿El juego funciona en 32 Bits?

No. Lamentablemente los desarrolladores del juego no proporcionaron una version de 32 Bits

¿El juego funciona en Windows 7?

El mod que se utiliza para aumentar el rendimiento no funciona alli. Pero si lo eliminas, este mismo funciona correctamente

⚠️¿Por que el juego me crashea al iniciar?

Esto puede deberse a incompatibilidades con tu tarjeta gráfica o controladores desactualizados. Sigue estos pasos:

  • - Actualiza tus controladores gráficos a la última versión disponible.
  • - Desactiva tu antivirus y descomprime el juego nuevamente
  • - Cierra procesos en segundo plano y agregale mas memoria virtual al sistema operativo
