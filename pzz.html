PROJECT ZOMBOID LITE!

Bienvenido al proyecto de PROJECT ZOMBOID LITE! Una version modificada, pensada para aumentar el rendimiento en computadoras de bajos recursos

¿Se le pueden poner mods al juego?

Claro que si. Por defecto el juego trae mods de optimizacion para aumentar el rendimiento!

¿Cuanto pesa el juego?

El juego pesa 3.84 GB comprimido y 6.17 GB descomprimido

¿Por que hay archivos corruptos en el juego y por ende no me inicia?

Esto te puede ocurrir al descargar el juego con una conexion a internet mala o inestable. Mi recomendacion es que utilizes 'Free Download Manager' o 'Internet Download Manager' para descargarlo y que lo descomprimas utilizando la ultima version de 7-Zip

¿Por que el juego me crashea al iniciar?

Esto podria ser algun error con tu tarjeta grafica o tus drivers, te recomiendo actualizar tu driver de video a la ultima version y desactivar tu antivirus al ejecutarlo

¿Puedo crear servidores?

Si puedes crearlos yendo a la carpeta ZomboidData y ejecutando el "ProjectZomboidServer"

¿Que version de java utiliza?

Utiliza GraalVM 21 para 64 bits y el java original del project zomboid para 32 bits

¿En que version esta basada?

Esta basada en la version 0.32.1

¿Por que mis antivirus detectan el juego como virus?

Esto se debe a los cracks y modificaciones que se realizan. Te recomiendo desactivar tu antivirus, ya que el crack engaña al juego para permitir su uso gratuito, lo cual es detectado por los antivirus. Para obtener más información, te sugiero leer el siguiente texto: Los antivirus y las detecciones.
